# 🖼️ Local AI Image Generator - 模型下载与汉化指南

## 📥 模型下载方法

### 方法1：内置模型管理器（推荐）
启动应用后，在界面中：

1. **打开"模型管理器"标签页**
2. **内置预置模型库**，包含：
   - SDXL高质量模型（6.6GB）
   - SD 1.5轻量模型（2.1GB）  
   - GGUF量化模型（3.8GB，CPU友好）

3. **一键下载**：
   - 直接点击模型卡片上的下载按钮
   - 自动从Hugging Face镜像源下载
   - 显示下载进度和速度

### 方法2：外部URL导入
1. **获取模型下载链接**（来自Hugging Face等平台）
2. **在模型管理器中"添加模型"**
3. **粘贴URL**自动下载

### 方法3：本地文件导入
1. **下载模型文件**（支持格式：`.safetensors`, `.gguf`, `.ckpt`）
2. **拖拽文件到** `app/models/` 文件夹
3. **或使用** "导入本地模型" 功能

### 🔥 推荐模型推荐

#### 高质量模型（适合显卡）
- **Juggernaut XL v9 Lightning** (6.6GB)
  - 用途：照片级真实感图像
  - 特点：4-8步闪电生成
  
- **DreamShaper XL Lightning** (6.6GB)  
  - 用途：幻想艺术、插画、风格化图像

#### 轻量模型（适合CPU/低显存）
- **DreamShaper 8** (2.1GB)
  - 用途：插画、动漫、半写实肖像
  - 显存：4GB+即可运行

- **SDXL Base Q4_0 GGUF** (3.8GB)
  - 用途：低显存或CPU友好
  - 格式：GGUF量化，适合CPU生成

## 🇨🇳 汉化说明

### 当前语言状态
**目前界面为英文**，但具备汉化基础：

#### ✅ 已支持汉化的部分
- 所有组件使用React国际化框架
- CSS样式系统采用现代设计系统
- 字体支持中文字符（使用系统字体栈）
- 主题系统支持深色/浅色模式

#### 🔧 汉化实现方案

##### 方案1：快速汉化（推荐）
修改 `app/frontend/src/App.css` 中的字体和颜色：

```css
/* 修改中文字体支持 */
:root {
  font-family: 'Microsoft YaHei', 'PingFang SC', 'Helvetica Neue', sans-serif;
}

/* 更新主题色为中文用户偏好 */
--md-sys-color-primary: #1976d2; /* 蓝色，更符合中文用户习惯 */
```

##### 方案2：完整汉化
需要创建语言文件：

1. **创建语言配置** `app/frontend/src/i18n/`
2. **翻译所有界面文本**
3. **集成多语言切换功能**

##### 方案3：混合界面
保持英文界面，仅汉化核心功能区域。

## 🚀 使用建议

### 新手推荐
1. **先安装 DreamShaper 8** (2.1GB)
2. **学习基本提示词写作**
3. **逐步尝试更高质量模型**

### 硬件配置建议
- **RTX 3060 12GB**：可运行所有SDXL模型
- **RTX 2060 6GB**：建议SD 1.5模型
- **CPU-only**：推荐GGUF量化模型

### 性能优化
- 使用适当模型：匹配硬件性能
- 调整生成步数：质量与速度平衡
- 启用GPU加速：大幅提升生成速度
