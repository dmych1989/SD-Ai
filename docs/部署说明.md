# Local AI Image Generator - 部署说明

## ✅ 已完成的部分

1. **Node.js 便携版** - 已安装到 `app\tools\node-win\` (v22.12.0)
2. **前端依赖** - 已通过 npm install 安装
3. **前端构建** - 已构建到 `app\dist\` 目录
4. **目录结构** - 已创建：
   - `app\models\` - 放置模型文件
   - `app\outputs\` - 生成图像输出目录
   - `app\backend\win\cuda\` - CUDA 后端目录
   - `app\backend\win\vulkan\` - Vulkan 后端目录
5. **启动脚本** - 已创建 `start.bat`

## ⚠️ 需要手动完成的部分

### 1. 下载 CUDA 后端（NVIDIA GPU）

**文件**: `sd-cuda.exe` 和 `stable-diffusion.dll`

**下载地址**:
```
https://github.com/leejet/stable-diffusion.cpp/releases/download/master-669-2d40a8b/sd-master-2d40a8b-bin-win-cuda12-x64.zip
```

**安装步骤**:
1. 下载上面的 ZIP 文件（约 335 MB）
2. 解压后，从 `bin\` 目录复制：
   - `sd-server.exe` → 重命名为 `sd-cuda.exe` → 放到 `app\backend\win\cuda\`
   - `stable-diffusion.dll` → 放到 `app\backend\win\cuda\`
   - 所有 `.dll` 文件 → 放到 `app\backend\win\cuda\`

### 2. 下载 Vulkan 后端（AMD/Intel GPU 或 CPU 回退）

**文件**: `sd-vulkan.exe` 和 `stable-diffusion.dll`

**下载地址**:
```
https://github.com/leejet/stable-diffusion.cpp/releases/download/master-669-2d40a8b/sd-master-2d40a8b-bin-win-vulkan-x64.zip
```

**安装步骤**: 同 CUDA，但放到 `app\backend\win\vulkan\` 目录

### 3. 下载模型文件

将 Stable Diffusion 模型文件（.safetensors, .gguf, 或 .ckpt）放到：
```
app\models\
```

**推荐模型**（可从这个项目列表选择）:
- SD 1.5 模型: https://huggingface.co/runwayml/stable-diffusion-v1-5
- SDXL 模型: https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0

## 🚀 启动方法

双击运行 `start.bat`，或手动执行：
```bat
cd D:\GitHub\Local-AI-Image-Generator
app\tools\node-win\node.exe scripts\serve.cjs
```

启动后访问: http://localhost:1420

## 📝 备注

- 如果只有 CPU（没有 NVIDIA GPU），可以跳过 CUDA 后端，使用 CPU 模式（会较慢）
- 模型文件较大（通常 2-8 GB），请确保有足够的磁盘空间
- 生成图片时，512x512 分辨率，20 步：
  - CUDA GPU: ~10 秒
  - CPU: ~150-300 秒

## 🔧 故障排除

如果遇到问题：
1. 运行 `scripts\reset.ps1` 重置环境（保留模型和输出）
2. 检查 `app\backend\win\cuda\` 或 `app\backend\win\vulkan\` 是否有正确的 exe 和 dll 文件
3. 确保模型文件完整（不完整的模型会导致加载失败）
